Invitacion BotCore.User.GestorInvitaciones.AlmacenarInvitacion ( string  nombreTempUsuario)

Metodo que crea la Invitacion y la envia al destinatario especificado. Se trata de una implementación de patrón Creator, ya que considerando la muy estrecha relación entre esta clase y la clase Invitacion, es razonable que la responsabilidad de instanciar esa clase recaiga en este Gestor, de forma que el resto de clases quedan libres de la responsabilidad de crear objetos Invitacion.

Parámetros
nombreTempUsuarioel nombre provisorio con el que se almacena el usuario.
Devuelve
La invitación creada.
